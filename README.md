## 2. 项目内容

本项目基于论文《Exploring the Controlling Factors of Watershed Streamflow Variability Using Hydrological and Machine Learning Models》，聚焦于滦河流域径流变化的主控因子分析。

*   **核心方法**：利用数据库流域模拟数据，基于R语言，将XGBoost与RF模型对径流量的预测值与实际值进行拟合分析。通过四个统计学指标(R^2、NSE、KGE、PBIAS）的量化对比，证明了XGBoost模型在整体预测性能上优于RF模型
*   **代码仓库**：[619890876-lgtm.github.io](https://github.com/619890876-lgtm/619890876-lgtm.github.io)
*   **研究方向**：水文水资源 / 流域径流变化模拟

## 3. 研究背景

传统水文研究中，径流模拟通常依赖物理机制明确的水文模型（如 SWAT），但其参数繁多、校准耗时且计算成本较高。纯粹的数据驱动模型虽然灵活，却又缺乏物理约束。Ding et al. (2025) 尝试将物理水文模型的特征与机器学习算法进行整合，通过提取 SWAT 模型的输入和输出数据构建了极端梯度提升（XGBoost）和随机森林（RF）两种算法模型并进行严格测试，发现：

1. 机器学习方法能够与物理水文模型（SWAT）有效结合，在复杂流域的径流模拟中展现出卓越的拟合与预测能力。
2. 相比于 RF 模型，XGBoost 在所有核心水文评估指标（$R^2$、NSE、KGE、PBIAS）上均表现出更优的径流模拟性能。
3. XGBoost 模型对径流存在轻微的低估（PBIAS = -1.445%），而 RF 存在轻微高估；两者在日常流量模拟上表现优异，但在大流量极端值预测上因数据频次较低而性能有所衰减。
4. 这种整合方法既保留了水文学的物理机制，又发挥了机器学习的高效性，避免了传统模型繁琐的参数调整。


## 4. 本项目复现目标

1. 复现论文核心机器学习建模流程：基于 ESBU 数据的训练集/测试集划分（70%/30%）、XGBoost 与 Random Forest 回归模型训练，以及水文统计指标的计算。
2. 复现关键图表：实际值与预测值的模型拟合散点图（包含 1:1 对角参考线及 $R^2$、NSE 等指标注释）。
3. 实现完整可复现的分析流程，使用 Python（如 `scikit-learn`、`xgboost`、`matplotlib`）配合 Jupyter 环境保证运行环境的一致性。
4. 复现并验证论文关于“XGBoost 在融合物理机制的水文径流模拟中性能优于 RF 模型”的核心结论。
